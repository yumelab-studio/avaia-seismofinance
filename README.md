# SeismoFinance: Predicting Market Reactions to Earthquakes

## Overview
This project explores how seismic events influence financial markets, focusing on insurance stock price reactions in Japan. We combine earthquake data with financial time-series data to identify patterns and potential predictive signals.

## Dataset
- Earthquake data: USGS
- Stock data: Yahoo Finance
- Merged dataset: 917 aligned samples
